CIMA-X pipeline deployed to namespace {{ .Values.namespace }}.

== Orthanc ==
  DICOM receiver:  {{ (index .Values.orthanc.nodePorts "dicom") }} (NodePort)
  Web UI / REST:   {{ (index .Values.orthanc.nodePorts "http") }} (NodePort)

  Web UI:    http://<node-ip>:{{ (index .Values.orthanc.nodePorts "http") }}/app/explorer.html
  Upload:    curl -u user:pass -X POST http://<node-ip>:{{ (index .Values.orthanc.nodePorts "http") }}/instances --data-binary @file.dcm

  If the orthanc-credentials secret is not already set up, create it with:
    kubectl create secret generic orthanc-credentials \
      --from-literal=users.json='{"RegisteredUsers": {"admin": "changeme"}}' \
      -n {{ .Values.namespace }}

== Samba ==
  Share name: {{ .Values.samba.shareName }}
  Connect:    smb://<username>@<node-ip>/{{ .Values.samba.shareName }}

  If the samba-credentials secret is not already set up, create it with:
    kubectl create secret generic samba-credentials \
      --from-literal=username=<user> \
      --from-literal=password=<pass> \
      -n {{ .Values.namespace }}

== xnat-ingest (sort / upload / associate) ==
  All three pods start with `sleep infinity`. Run manually with kubectl exec:

  Sort:
    kubectl exec -it sort -n {{ .Values.namespace }} -- \
      xnat-ingest sort-cli /data/orthanc-storage /data/staging/sorted --recursive

  Upload:
    kubectl exec -it upload -n {{ .Values.namespace }} -- \
      xnat-ingest upload-cli /data/staging/sorted $XINGEST_SERVER

  Associate:
    kubectl exec -it associate -n {{ .Values.namespace }} -- \
      xnat-ingest associate-cli /data/cima-export /data/staging/sorted \
        --associated-files 'medimage/vnd.siemens.syngo-mr.xa.rda' \
          '{PatientName}/**/*.rda' \
          '.*\.(?P<id>\d+)\.\d+\.\d+\.(?P<resource>[^.]+)'

  If the xnat-credentials secret is not already set up, create it with:
    kubectl create secret generic xnat-credentials \
      --from-literal=server=https://xnat.example.org \
      --from-literal=username=<user> \
      --from-literal=password=<pass> \
      -n {{ .Values.namespace }}
